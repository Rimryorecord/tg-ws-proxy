## 📌 Overview

TG WS Proxy is a local proxy tool that uses SOCKS5 and WebSocket (WSS) to accelerate Telegram connections, bypass network restrictions, and improve overall stability.

It runs entirely on your local machine and does not rely on third-party servers, ensuring full privacy and security.

---

## ✨ Features

- 🔐 Local SOCKS5 proxy for secure routing  
- ⚡️ WebSocket (WSS) for faster connections  
- 🌐 Bypass censorship, firewalls, and ISP restrictions  
- 🛡 Full support for MTProto end-to-end encryption  
- 🧠 Automatic Telegram Data Center (DC) detection  
- 🔀 Smart routing via official Telegram domains  
- 🔁 Automatic fallback to TCP if WSS is unavailable  
- 📉 Reduced latency and improved connection stability  
- 🖥 Cross-platform support (Windows / macOS / Linux)  
- 🧭 GUI (system tray) + CLI mode  
- ⚙️ Easy integration with Telegram Desktop  
- 📊 Configurable settings and detailed logging  
- 🧑‍💻 Open-source with automated builds  

---

## 🚀 How It Works

TG WS Proxy intercepts Telegram traffic and routes it through a local SOCKS5 proxy using WebSocket over TLS.

This allows you to:

- Improve connection speed  
- Bypass network restrictions  
- Maintain full encryption and privacy  

If WebSocket transport is unavailable, the application automatically switches to a direct TCP connection.
